登入 Canvas
Due 透過你的 Canvas 登入狀態同步資料，不需要額外設定或 Token。
  • 1登入 Canvas ↗ (基本上所有人應該都是登入的，如果沒有你可能沒在上課😭)
  • 2登入後，Due 會在你每次打開Canvas時自動同步你的所有課程與作業
  • 3點擊工具列或進入 Dashboard 都可以看到你的作業
